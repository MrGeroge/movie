电影馆

放映文化的承重体——FIRST青年电影馆

“电影需要观看”一直是FIRST青年电影展践行放映文化的根本理念，FIRST青年电影馆就是由此衍生出的承重体，竭力建构青年电影放映品牌，并通过议题的发酵，营造开放性的对话场域、建构多文化的评价语境。
此前，FIRST青年电影馆已经举办了三季放映沙龙，在延续“观看的时代”这一基本放映主题的基础上，尝试在不同的策展纬度中对“青年电影”和“电影处女作”进行了探讨，比如搭配世界影电影大师的早期作品作为镜像与青年电影形成互文，并邀请各路文化界的前辈解读电影的纵深，与观众深入对话。

支持机构 :朝阳区文化馆

全国文化体制改革试点单位，以 “百姓的文化事业，百分之百的努力”为宗旨；从保障民众的文化权益和百姓的文化需求出发，坚持公共文化优先，用创意实现发展，在实践中不断拓展文化服务空间和创新文化服务方式。其中“9个剧场”品牌的建立，把文化馆的文化触角深入到北京市主流演出市场，以优秀的文化环境，一流的场地设施和认真的从业精神，推动了北京小剧场戏剧运动及非职业戏剧运动发展，形成戏剧东移之势。

资源设备

拥有2个可以播放胶片和数字拷贝、3D电影的放映厅，共140个座位。依托朝阳区文化馆的多样资源，馆内还拥有9个风格各异的戏剧演出剧场、咖啡馆和露天放映场所，方便观众进行选择和休憩。

最新活动：

10月24日，FIRST青年电影馆新一季“在场”沙龙活动开启，放映本年度影展提名影片及获奖影片，也是自7月西宁后，首次大规模放映， 由于放映所选取的青年作品在表现手法、现实思考、艺术价值上，都呈现丰富的多样性和解读空间，提供了构建不同的对谈议题的可能性。本次的观影沙龙将根据不同放映片目，拟定针对性的主讲议题，邀请电影主创及相关领域的资深从业者、专家、学者参与映后沙龙。沙龙将打破放映片目的内容局限，突破放映厅的场域界限，从核心命题衍发进行深度交流、广度发酵。

何为“在场”沙龙

由影展“立场”衍生出的电影思辨方式，带有无法避免的局限性，而削弱单方主导的最佳方式无疑是让多样化的身份“在场”。2015年“在场”电影沙龙，FIRST青年电影展将与《电影世界》联合主办，让策展人、媒体、嘉宾、观众的分析和批评同时存在，各方“身份”彼此消解后，让讨论回归客观，还原对青年电影作品和青年创作群体的清醒解读，不美化电影的真实，而是怀着美好的期待，去正视青年电影的当下。
