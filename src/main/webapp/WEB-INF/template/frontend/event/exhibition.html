巡展介绍

策展人的话：

从2010年开始，我们在非电影节期间有了自发的放映行为，那个时候还不正规，形式时而松散时而紧张，至今4年多的时间。按照剧作的原理来讲，我们的放映是缺乏动机的，因为大部分时候没有一个十分明确的目的，就像在一个促狭的场合，小伙子总要对害羞的姑娘说点什么一样，我们总觉得需要在非电影节期间做点离电影更近的营生，于是放映就这么开始了。
偶尔，还是会怀疑我们的放映到底能够起到什么效果，当然也会质疑自己对于电影本身的审美趣味，我们的放映偶尔也会有技术故障，很多时候，我们自己都怀疑可能是因为缺乏一个足够的动机，而这份怀疑就像是所有处在事业起步阶段的人遇到的一样，充满未知却又蕴含无限可能，直到后来我们发现，怀疑其实是一份礼物。
如今，我们依然对一切持怀疑态度，难道不能上映的独立电影就真的是好电影么？不一定。艺术院线就优于商业院线么？不一定。资本就一定是腐朽的么？还是不一定。无论是赞美还是批判，我们只希望能够探讨的更多，好和坏从来都是相对而言，而多元的共生却能带来无限的可能，哪怕它危险丛生。所以去他的动机吧，谁让我们是主动放映的呢。
2015年主动放映中国巡展走过百站，放映场次超过270场。其中，大规模城市巡展7站，分别是长春站、宜昌站、厦门站、重庆站、洛阳站、济南站、石家庄站；学校巡展65站，同时还有开辟了独立书店放映、青旅放映、咖啡厅放映、艺术空间放映、露天放映、livehouse放映、商业影院放映、剧场放映等多元放映形式30余站。放映范围覆盖台湾、香港、新疆、西藏、内蒙、广西、江苏等在内的28个省级行政区，重庆、北京、上海、天津在内的40多个城市。社会新闻媒体介入报道6家，学校、社团、机构自媒体主动报道超过80家，宣传覆盖人群超过300万人次。
